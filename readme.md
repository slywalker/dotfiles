# Usage

First, install homesick

```
$ gem install homesick
$ cd ~ 
$ homesick clone git@github.com:slywalker/dotfiles.git
$ homesick symlink dotfiles
```

After, brew installed [Homebrew â€” The missing package manager for OS X](http://brew.sh/)

```
$ brew bundle
$ gem install bundler
$ bundle install
$ composer global install
$ pear-install
```
